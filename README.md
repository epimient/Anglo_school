# Anglo Colombia School — Website Project

Este es el repositorio oficial del sitio web del **Anglo Colombia School**, una institución educativa bilingüe de prestigio ubicada en Barranquilla, Colombia.

## 🚀 Características
- **Diseño Premium**: Interfaz moderna, elegante y totalmente responsiva.
- **Identidad Corporativa**: Integración de logos oficiales y paleta de colores institucional.
- **Interactividad**: Animaciones fluidas, scroll adaptativo y elementos dinámicos con JavaScript.
- **Documentación**: Acceso directo desde el menú a documentos institucionales (Manual de Convivencia, Reglamento Interno).
- **Servicios**: Secciones dedicadas a plataformas académicas (Phidias, Biofood) con acceso rápido.

## 🛠️ Stack Tecnológico
- **Frontend**: HTML5, Vanilla CSS3, JavaScript (ES6+).
- **Framework**: Bootstrap 5.3.3 (CDN).
- **Iconografía**: Bootstrap Icons.
- **Tipografía**: Google Fonts (Playfair Display & Inter).
- **Gráficos**: Formatos SVG para logos y optimización de imágenes JPG/PNG.

## 📂 Estructura del Proyecto
```text
Anglo_school/
├── index.html          # Punto de entrada principal
├── css/
│   └── styles.css      # Estilos personalizados y sistema de diseño
├── js/
│   └── main.js        # Lógica de animaciones e interactividad
├── images/             # Activos visuales y fotos optimizadas
├── favicon.svg         # Icono de la pestaña del navegador
└── README.md           # Documentación del proyecto
```

## 💻 Desarrollo Local
No se requiere un servidor complejo. Simplemente abre `index.html` en cualquier navegador moderno o utiliza una extensión de "Live Server" para una mejor experiencia de desarrollo.
