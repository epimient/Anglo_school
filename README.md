# 🎓 Anglo Colombia School — Sitio Web Institucional

> *"La excelencia educativa merece una presencia digital a su altura."*

---

## 📋 Descripción

Repositorio oficial del sitio web del **Anglo Colombia School**, institución educativa bilingüe de prestigio en Barranquilla, Colombia. Este proyecto representa la puerta de entrada digital para estudiantes, padres y comunidad educativa.

---

## ✨ Características Destacadas

| Característica | Descripción |
|----------------|-------------|
| 🎨 **Diseño Premium** | Interfaz moderna, elegante y 100% responsiva |
| 🏫 **Identidad Corporativa** | Logos oficiales y paleta de colores institucional integrada |
| ⚡ **Interactividad** | Animaciones fluidas, scroll adaptativo y elementos dinámicos |
| 📚 **Documentación** | Acceso directo al Manual de Convivencia y Reglamento Interno |
| 🔗 **Servicios** | Plataformas académicas (Phidias, Biofood) con acceso rápido |

## 🚀 Inicio Rápido

### Opción 1: Navegador Directo
```bash
# Simplemente abre el archivo en tu navegador favorito
xdg-open index.html          # Linux
open index.html              # macOS
start index.html             # Windows
```

### Opción 2: Live Server (Recomendado para desarrollo)
```bash
# Con VS Code: Instala la extensión "Live Server"
# Con Python:
python -m http.server 8000   # Accede en http://localhost:8000
```
